TinyLink is a lightweight, full-stack URL shortener built with **Node.js, Express, PostgreSQL, and Vanilla JS**.  
It provides link management, redirects, analytics, and can be deployed to cloud platforms like **Render + NeonDB**.

Live Demo: https://tinylink-2bet.onrender.com/ 
Repository: https://github.com/Amol-Deshmane/tinylink

---

##  Features

-  Create short, shareable URLs  
-  Custom or auto-generated short codes  
-  Dashboard to view, search, delete links  
-  Automatic redirect via `/:code`  
-  REST API support  
-  Environment-variable based config  
-  Deployable on Render + Neon
